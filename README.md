Arch Installer Script
========

A simple Archlinux installation script written in bash.

Get Started
------------------
Boot on Archlinnux live.

Load keyboard layout (fr, en, de...):
`$ loadkeys fr`

Install git:
`$ pacman -S git`

Clone arch-installer:
`$ git clone https://github.com/m4d4rchy/arch-installer`

Run script:
`$ chmod +x`
`$ ./arch-installer`
